\chapter{Análise Teórica}

Esta etapa consiste na modelagem matemática do filtro passa-faixa de segunda ordem com configuração de realimentação múltipla (MFB).

\section{Função de Transferência}

Aplicando a Lei de Kirchhoff das Correntes (LKC) no domínio da frequência ($s$) e considerando o amplificador operacional ideal, a função de transferência é definida por:

\begin{equation}
	H(s) = \frac{V_o(s)}{V_i(s)} = \frac{-\frac{1}{R_1 C} s}{s^2 + \frac{2}{R_3 C} s + \frac{R_1 + R_2}{R_1 R_2 R_3 C^2}}
\end{equation}

Para os valores nominais utilizados ($R_1 = 470\,\Omega$, $R_2 = 470\,\Omega$, $R_3 = 47\,k\Omega$ e $C = 100\,nF$), a Tabela \ref{tab:valores_teoricos} apresenta os valores calculados via software \cite{maxima}.
